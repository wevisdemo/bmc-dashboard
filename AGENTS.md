# BMC Dashboard

- Main tech stack are Svelte, Typescript, Bits UI, and TailwindCSS
- Always use pnpm as the package manager
- Prefer using tailwind css for styling, and avoid using custom css as much as possible
- Use Svelte's built-in features and avoid using external libraries unless necessary
- Activate Svelte skill before start making changes, or planning of making changes, to svelte file
- After finishing any task, run the following commands:
  - Check type with `pnpm run check`
  - Lint with `pnpm run lint`, all errors and warnings must be fixed
  - Format code with `pnpm run format` before declaring task as done
- Human will get in the loop and edit some file along the way. If you spot it, please respect those changes.
